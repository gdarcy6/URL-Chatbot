# URL Chatbot

This is a local chatbot designed to seamlessly interact with web pages! 
The chatbot enables users to input a URL of any article or page and ask insightful questions related to its content. 
Built using Langchain, this chatbot leverages advanced packages like HuggingFace to interpret the page and Ollama to run the Llama3 large language model entirely locally!
